# Item texture drop-in folder

Farmer's DragN item models are configured like Farmer's Delight's standard generated-item setup:

- model path: `assets/farmersdragn/models/item/<item_id>.json`
- texture key: `"layer0": "farmersdragn:item/<item_id>"`
- expected image path: `assets/farmersdragn/textures/item/<item_id>.png`

To add/update an icon, just place a PNG named after the item id in this folder.

Examples:

- `goat_cheese_cut.png`
- `dog_food.png`
- `cat_food.png`
- `farmers_dragn_mini.png` (creative-tab icon item)

Recommended size: 16x16 (or 32x32 for higher detail).
